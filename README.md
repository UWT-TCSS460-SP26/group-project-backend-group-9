# TCSS 460 — Group Project Backend

Express + TypeScript API for the TCSS 460 group project.

## Deployed URL

https://tcss460-team-9-api-72a4ffcce368.herokuapp.com

## Auth Tokens

Tokens can be acquired at the [Token Playground](https://tcss460-token-playground.onrender.com/)
under the `group-9-api` audience.

## Endpoints

Endpoints are documented in the [API docs]([https://tcss460-token-playground.onrender.com/](https://tcss460-team-9-api-72a4ffcce368.herokuapp.com/)api-docs)

## CORS

CORS allow-list:

- `http://localhost:3000`

To have an origin added to the allow-list, submit an issue
at our [issue reporting endpoint]([https://tcss460-token-playground.onrender.com/](https://tcss460-team-9-api-72a4ffcce368.herokuapp.com/)report)

## Reporting Bugs
